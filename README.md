# Multi Agent Financial Report Generator with Agents

## Overview

This project is an **intelligent multi-agent system** designed to autonomously generate **comprehensive, data-driven financial performance reports**. By leveraging multiple specialized agents, the system transforms a single query into a complete, actionable report.

## Features

* **Agent-Based Pipeline**: Converts a single user query into a full financial report automatically.
* **Meta-Prompting**: Enhances planner decisions and optimizes tool selection for better results.
* **Integrated Tools**:

  * **Web Search Agents**: Collect relevant financial data in real-time.
  * **OCR Agents**: Extract information from documents for end-to-end financial insights.

## Benefits

* Rapid, automated financial reporting.
* Data-driven insights without manual intervention.
* Scalable framework adaptable to various financial domains.
